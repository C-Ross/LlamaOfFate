# Prompt Templates

Go templates used to build LLM prompts for Fate Core narration and action parsing.

## Fate Core SRD Attribution

The templates in this directory contain paraphrased or near-verbatim rules from the **[Fate Core System Reference Document](https://fate-srd.com/fate-core)**, a product of Evil Hat Productions, LLC. Covered rules include the four action types, aspect and advantage mechanics, conflict resolution, stress and consequences, challenges, and GM principles.
